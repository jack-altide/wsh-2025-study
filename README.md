# WSH 2025 学習プラン

[Web Speed Hackathon 2025](https://github.com/CyberAgentHack/web-speed-hackathon-2025) のコードを理解するための学習タスク管理リポジトリです。

## 学習対象の技術スタック

### フロントエンド
- TypeScript / React 19 / React Router 7
- Zustand（状態管理）
- UnocCSS（スタイリング）
- HLS.js / Video.js（動画再生）

### バックエンド
- Node.js / Fastify 5
- SQLite + Drizzle ORM
- SSR（サーバーサイドレンダリング）

### 開発ツール
- pnpm（パッケージ管理）
- Webpack 5（バンドル）
- Playwright（テスト）
