# 学習進捗確認コマンド

現在の学習進捗を確認します。

## 実行内容

1. `notes/` フォルダ内のファイルを確認
2. 各Phaseの学習状況を表形式で表示:
   - Phase番号
   - テーマ
   - 完了したトピック数
   - 進捗率

3. 次に学習すべきトピックを提案

4. 学習のペースについてフィードバックを提供

## 出力形式

```
## 学習進捗レポート

| Phase | テーマ | 進捗 |
|-------|--------|------|
| 1 | 言語とツールの基礎 | ○/○ |
| 2 | フロントエンド基礎 | ○/○ |
...

### 次のステップ
- 推奨: [次のトピック]
```
