# 学習トピック開始コマンド

新しい学習トピックの練習問題に取り組みます。
**前提**: `/explain` で概念を理解済みであること。

## 引数
- `$ARGUMENTS` - 学習したいトピック名（例: "Step 2-1 HTML/CSS"）

## 学習の流れ
1. まず `/explain` で概念を理解する
2. その後 `/learn` で練習問題に取り組む

## 実行内容

### 0. Issue確認（最重要）
- まず `gh issue list` で対応するIssueを確認する
- Issueの「学習内容」チェックリストを全て把握する
- **Issueの全項目をカバーするまで学習を完了としない**
- 学習の途中で進捗をチェックリストと照合する

### 1. 練習問題
- `exercises/typescript-basics/todo.ts` のタスク管理アプリを拡張する形で練習問題を出題する
- 新しい概念をタスク管理アプリに組み込む練習にする
- 穴埋め形式やコメントで指示を出し、ユーザーが自分で書く部分を残す
- 各ステップで学んだ概念を実際に手を動かして確認できるようにする
- 回答後にコンパイル・実行して動作確認する
- トピックによってはタスク管理アプリの拡張が難しい場合、別ファイルでもOK

### 2. 理解度確認
理解度確認のための質問を2-3個提示してください

### 3. 完了チェック
- Issueの「学習内容」チェックリストと照合し、未カバーの項目がないか確認する
- 未カバーの項目があれば、追加で学習を行う
- 全項目をカバーしたことを確認してから学習ノートを作成する

### 4. ノート作成
学習ノートを `notes/` に作成してください

トピック: $ARGUMENTS
