# 学習トピック開始コマンド

新しい学習トピックを開始します。

## 引数
- `$ARGUMENTS` - 学習したいトピック名

## 実行内容

1. 指定されたトピックについて、以下の構成で説明してください：
   - **概要**: トピックの基本的な説明
   - **なぜ重要か**: このプロジェクトでの役割
   - **基本概念**: 押さえておくべき重要な概念
   - **コード例**: 実際の使用例
   - **WSH 2025での使われ方**: Web Speed Hackathonでの具体的な活用

2. 理解度確認のための質問を2-3個提示してください

3. 学習ノートを `notes/` に作成してください

トピック: $ARGUMENTS
